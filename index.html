Project

TEAMWORK MAKES THE DREAM WORK

CT40 Digital Wishing Tree

CT40 Wishing Tree website was created to celebrate
the 40th anniversary of Clementi Town secondary school.
This website aims to take in students’ wishes for the school
and display it on the digital Tree.
Every wish posted will be sent to the admin server for
verification before displaying it on the Tree.
For every 200 wishes added to the Tree,
the Tree will grow bigger and move on to the next stage.
This is to emphasize the school’s motto of “Aspire and Grow”.
